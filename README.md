# 🗂️ Registros_API

Aplicación web para la gestión de registros de ciudadanos, con validación de datos, integración de API (SOAP simulada y REST), y verificación.

## 🚀 Tecnologías
- Python
- Flask
- HTML + JavaScript
- Oracle DB
- API SOAP/REST
